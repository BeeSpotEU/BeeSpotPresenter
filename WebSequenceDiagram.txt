title Beespot presentatie ondersteuning

participant Server
actor Presentator
participant Beamer
actor Deelnemer

group Nieuwe sessie
Presentator->Server: Start nieuwe sessie
Server->Beamer: Toon pincode + url
Beamer->Beamer: Toont pincode + url
Deelnemer<-Beamer: Neemt pincode over
end 

group Deelnemer toevoegen
Deelnemer->Server: Start in sessie
Server->Deelnemer: Stuur formulier
note left of Deelnemer: Naam\n(postcode, huisnummer)
Deelnemer->Server: Verstuur formulier
Server->Presentator: Stuur aantal deelnemers
Server->Beamer: Stuur aantal deelnemers
Beamer->Beamer: Toon aantal deelnemers\nToon pincode
end

group Bijenstand toevoegen
loop response != klaar

Server->Deelnemer: Stuur formulier
note left of Deelnemer: Bijenstandnaam\nLocatie\nAantal volken vorig jaar
Deelnemer->Deelnemer:Invullen Bijenstandnaam
group Locatie Zoeken
alt adresveld
Deelnemer->Server: adres 
Server->Deelnemer: coordinaten
else sleep op map
Deelnemer->Server: coordinaten vanaf kaart
end
Server->Presentator: Stuur aantal Bijenstanden
Presentator->Presentator: Toon aantal deelnemers\nToon aantal bijenstanden
Server->Beamer: Nieuwe bijenstand locatie
Beamer->Beamer: Toon aantal deelnemers\nToon aantal bijenstanden\nToon pincode\nToon kaart met bijenstanden
end

end

group Dracht verwachting per bijenstand (van een specifieke deelnemer)

Presentator->Server: Start Drachtverwachting

Server->Beamer: Drachtverwachting slide

loop per bijenstand

Server->Deelnemer: Stuur formulier
note left of Deelnemer: Dracht hoeveelheid\n(Diversiteit)

Deelnemer->Server: Verstuur formulier

Server->Beamer: Stuur Drachtverwachting
Beamer->Beamer: Toon aantal bijenstanden\nToon barchat Drachtverwachting

end

end

group Uitleg basisdata
Presentator->Server: Start Uitleg basisdata

Server->Presentator: Open kaart met percelen
Server->Beamer: Open kaart met percelen
Server->Deelnemer: Open kaart met percelen
Presentator->Server: Klik blok aan
Server->Beamer: Zoom kaart op blok
Server->Beamer: Open view met planttypen en drachtplanten tabel
Server->Deelnemer: Open view met planttypen en drachtplanten tabel
Presentator->Server: Toon voedselindex kaart
Server->Presentator: Open voedsel indexkaart
Server->Beamer: Open voedsel indexkaart
Server->Deelnemer: Open voedsel indexkaart (zoom-in op eigen bijen stand)
Deelnemer->Deelnemer: Exploreren
end

group Vraag stellen, Opmerking maken
Deelnemer->Server: Vraag
note left of Deelnemer: Context, tekst, locatie
Server->Presentator: Vragen view
end

group Verschil met ingevuld
Presentator->Server: Start Verschillenkaart

Server->Presentator: Open hotspots
Server->Beamer: Open hotspots
Server->Deelnemer: Zoom drachtverwachting, toon verschillen in tabel
end

group Opbrengst per bijenstand
Presentator->Server: Start Opbrengst
Server->Beamer: Opbrengstenkaart slide
loop per bijenstand
Server->Deelnemer: Stuur formulier
note left of Deelnemer: Honing uit bijenstal gehaald (totaal vorig jaar)\nsuiker gevoerd (totaal vorig jaar)
Deelnemer->Server: Verstuur formulier
Server->Presentator: Aantal opgestuurd
Server->Beamer: Toon op kaart
end
end

group Kansenkaart
Presentator->Server: Toon kansenkaart
Server->Beamer: Kansenkaart
Server->Deelnemer: Dracht- en opbrengsten lagen 
Presentator->Server: Leeg maken
group uit balans
Server->Deelnemer: Kaart leeg
Deelnemer->Server: Uit balans
Server->Beamer: Heatmap
end
Presentator->Server: leegmaken
group kansen voor stal
Server->Deelnemer: Kaart leeg
Deelnemer->Server: Kansen
Server->Beamer: Heatmap
end

end
